# Privacy Policy

Spectator Zen for Twitch respects your privacy.

This extension does **not** collect, store, transmit, or share any personal data.

---

## Data Collection

Spectator Zen does **not** collect:
- Personal information
- Browsing history
- User activity
- Analytics or telemetry data

---

## Data Usage

The extension only uses Chrome's `storage` API to save **local user preferences**, such as:
- Whether Zen Mode is enabled
- Which layout options are selected

All settings are stored locally using Chrome’s extension storage and are **never transmitted** outside the user’s browser.

---

## Third-Party Services

Spectator Zen:
- Does **not** use third-party analytics
- Does **not** include tracking scripts
- Does **not** communicate with external servers

---

## Permissions Explanation

- **`https://*.twitch.tv/*`**  
  Required to apply layout changes on Twitch pages.

- **`storage`**  
  Used to save user preferences locally.

- **`commands`**  
  Used to enable keyboard shortcuts.

No other permissions are requested.
